Данное веб-приложение было реализован в контексте научно-социального проекта: 

> «Экскурсия с доставкой»: \
> Приложение виртуальных экскурсий/интерактивного гида по культурному и повседневному Челябинску

Данный репозиторий представляет собой [лендинговую страницу](./landing/) и [основное приложение](./app/) проекта.

### Используемые библиотеки

- **[MapLibre](https://github.com/maplibre/maplibre-gl-js)**: Интерактивная карта (маркеры, popup)
- **[FontAwesome](https://github.com/FortAwesome/Font-Awesome)**: Безупречные иконки
