## CSS
Каждый компонент имеет свой `css`-файл. В компоненте рекомендуются использовать псевдокласс `:root` для иниализация переменных таких как: цвета, отступы, скругления, размер шрифта и т.д.

Используйте комментарии для отделения зон ответственностей.
#### Пример
```css
:root {
  --background: #FFF;
  --backgroubd-alt: #F5F5F7;

  /* -- Typegraphy -- */
  --size-title: 14px;
  --size-subtitle: 14px;
  --color-title: #121420;
  --color-subtitle: #A5A7AC;
}
```
