# Стиль написания проекта

Чтобы проект был читабельным, легко поддерживаемым и удобен для других разработчиков - было принято установить единный стиль написания кода проекта:

## Общие рекомендации

- Отступы - 2 таба
- Названия функций - PascalCase: `FunctionName()`
- Название переменных - camelCase `let personValue = -1`

```tsx
function NewComponent() {
  return (
    <p>Hello, World!</p>
  );
};

export default NewComponent;
```
